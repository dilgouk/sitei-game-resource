# sitei-game-resource

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态资源的集中存储与分发使用。

## 项目简介

本仓库存放了一系列独立的 HTML 页面，这些页面可用于各类场景下的前端展示、游戏交互、工具界面等用途。所有页面均为纯前端实现，无需后端支持，可直接在浏览器中运行。

## 目录说明

```
.
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 共用资源文件（样式、脚本、图片等）
├── docs/           # 文档与说明
└── README.md       # 本文件
```

- **pages/**：每个子目录对应一个独立页面，包含其完整的 HTML、CSS、JavaScript 文件。
- **assets/**：存放多个页面可能共用的资源，如通用样式库、工具脚本、占位图片等。
- **docs/**：包含使用说明、示例截图或开发笔记。

## 页面归档说明

- 每个页面以独立子目录形式存放，目录名即为页面标识。
- 页面内所有资源均使用相对路径引用，确保可直接在本地或静态服务器上运行。
- 归档页面可能不定期更新、修复或移除，不保证长期可用性。
- 仓库仅提供静态页面存档，不包含任何后端服务或动态内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `pages/` 下任意子目录中的 `index.html` 文件即可查看对应页面。
3. 也可将整个仓库部署到任意静态文件服务器，通过 URL 访问对应页面。
